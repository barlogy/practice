## Описание проекта

Цель проекта — реализовать алгоритмы, которые угадывают загаданное компьютером целое число в диапазоне 1–100 за минимальное число попыток.

## Задача

Написать программу (и несколько стратегий), которая с помощью подсказок "больше/меньше" находит загаданное число. В репозитории реализованы, например, следующие подходы:

- случайная стратегия (генерирует числа случайно),
- стратегия бинарного поиска (последовательно сужает диапазон).

## Метрика качества

Качество алгоритма оценивается по средней длине (количеству попыток) угадывания за 1000 запусков.

## Запуск и установка

Рекомендуется использовать виртуальное окружение. Пример для macOS / zsh:

```bash
# создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate

# установить зависимости
pip install -U pip
pip install numpy
```

Запустить тест сравнения алгоритмов:

```bash
python src/PYTHON-8/guess_game/game_v2.py
```

[Файл с реализацией](game_v2.py)

## Результаты и выводы

В программе реализована функция `score_game`, которая прогоняет выбранный алгоритм по 1000 случайных чисел и возвращает среднее количество попыток. Ожидаем, что бинарный поиск будет значительно эффективнее случайной стратегии.
